Use Beads with Windsurf through a project rules file.

```bash
bd setup windsurf
bd setup windsurf --check
```

The setup command creates `.windsurf/rules/beads.md` with Beads workflow guidance.

## Remove

```bash
bd setup windsurf --remove
```
